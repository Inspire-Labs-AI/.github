### How it works

A continuous loop across every surface: **Research → Decide → Execute → Learn.**
AI workflows handle execution; human operators make the calls that matter.
Not chatbots. Workers.

### Open source

The name is the intent: Inspirelabs is here to **inspire engineering**. So we
release the tools we build for ourselves, MIT-licensed and developed in the
open, for any team that wants to build the same way. Everything here runs inside
Inspirelabs first; if it earns its place, we work to make it useful everywhere.

| Project | What it is |
| --- | --- |
| **[Synapse](https://github.com/Inspire-Labs-AI/synapse)** &nbsp;`Go · MIT` | AI-powered codebase-intelligence engine. Builds an AST dependency graph fused with a vector semantic layer, so you can visually explore, chat with, and map capability reuse across a codebase. |
| **[iForge](https://github.com/Inspire-Labs-AI/iforge)** &nbsp;`TypeScript · MIT` | Claude Code, run as a disciplined engineering team: product, architecture, design, review, QA, security, and release, all driven by slash commands. |
